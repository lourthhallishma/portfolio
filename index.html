Sentiment Analysis of Book Reviews

This project performs sentiment analysis on Amazon Kindle book reviews using Natural Language Processing (NLP) and Machine Learning. The models implemented include Random Forest Classifier, Gradient Boosting Classifier, Multinomial Naïve Bayes, and Support Vector Machine (SVM).

Python ML models and NLP Pandas Scikit-learn

COVID-19 & Pneumonia Classification using CNN

This project implements a Convolutional Neural Network (CNN) model to classify chest X-ray images into three categories: COVID-19, Pneumonia and Normal. The goal is to assist in the early diagnosis of respiratory illnesses using deep learning and medical imaging.

Python Convolutional Neural Networks Pandas Tensorflow

AI-powered Fashion Stylist
